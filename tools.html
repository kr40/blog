About This Tools Page

Welcome to my collection of useful tools! This page houses a variety of utilities that I've created or adapted because I find them interesting and potentially useful for everyday tasks.

As a cybersecurity enthusiast and developer, I often need small, purpose-built tools for specific tasks. Rather than searching for these tools online each time, I've decided to build and host them here for easy access.

The current toolkit includes:

  • Markdown Editor - A simple editor for creating blog posts with front matter, perfect for static site generation.
  • Calculator - A basic calculator for quick calculations.

I'll be adding more tools over time as I develop them. If you have suggestions or find these tools useful, feel free to reach out!
